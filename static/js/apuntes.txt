javascript

const resizeBakgroundImg = () => {
  const height = window.innerHeight - navbar.clientHeight;
  welcome.style.height = `${height}px`;
};

    resizeBakgroundImg: Define una función llamada resizeBakgroundImg que se utiliza para ajustar la altura del elemento welcome en función de la altura de la ventana y la altura de la barra de navegación (navbar).
    height: Calcula la altura restando la altura de la barra de navegación (navbar.clientHeight) a la altura de la ventana (window.innerHeight).
    welcome.style.height: Establece la altura del elemento welcome utilizando la propiedad CSS height. La altura se establece en píxeles utilizando la sintaxis ${height}px.

navbarToggle.ontransitionend = resizeBakgroundImg;
navbarToggle.ontransitionstart = resizeBakgroundImg;

    navbarToggle.ontransitionend: Asigna la función resizeBakgroundImg al evento transitionend del elemento navbarToggle. Esto significa que cuando la transición de navbarToggle finaliza, se llama a la función resizeBakgroundImg.
    navbarToggle.ontransitionstart: Asigna la función resizeBakgroundImg al evento transitionstart del elemento navbarToggle. Esto significa que cuando la transición de navbarToggle comienza, se llama a la función resizeBakgroundImg.
    window.onresize: Asigna la función resizeBakgroundImg al evento resize de la ventana. Esto significa que cuando se redimensiona la ventana, se llama a la función resizeBakgroundImg.
    window.onload: Asigna la función resizeBakgroundImg al evento load de la ventana. Esto significa que cuando la ventana y todos los recursos se han cargado completamente, se llama a la función resizeBakgroundImg.
